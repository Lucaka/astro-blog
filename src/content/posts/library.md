平常遇到、覺得值得收藏的前端函式庫整理，依用途分類方便日後查找。

## 地圖

在 Web 瀏覽器上顯示地圖：

- [Google Maps Platform](https://mapsplatform.google.com/)
- [OpenLayers](https://openlayers.org/)
- [Leaflet](https://leafletjs.com/)

## 圖表與時間軸

- 時間軸：[vue-ganttastic](https://infectoone.github.io/vue-ganttastic-homepage/?ref=madewithvuejs.com#/example)
- 量化圖表：[ECharts](https://echarts.apache.org/examples/en/index.html#chart-type-bar)、[Chart.js](https://www.chartjs.org/docs/latest/samples/bar/border-radius.html)
- [Vizzu](https://lib.vizzuhq.com/latest/#stories)：專為動畫設計的資料視覺化函式庫。特色：以動畫為核心、內建符合視覺化準則的預設樣式、自動彙整與過濾資料、以 HTML5 canvas 繪製、由 C++ 編譯成 WebAssembly、零相依。

## 動畫與視覺特效

- [party.js](https://party.js.org/)：用視覺特效（如彩帶、灑花）點亮使用者的操作體驗。
- [AnimXYZ](https://animxyz.com/)：可組合的 CSS 動畫工具組。
- [Whirl](https://whirl.netlify.app/)：只需極少功夫就能用上的 CSS loading 動畫集。
